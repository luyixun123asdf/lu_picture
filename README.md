### 企业级智能协同云图库平台 (Spring Boot, Redis, COS, AI, WebSocket)

**项目描述：** 设计并实现基于微服务架构的企业级智能云图库平台，包含公共图库、私有图库及团队共享图库三大核心模块。支持用户公开上传检索图片、管理员审核管理分析图片、个人用户私有图片批量管理/多维检索/编辑分析，以及企业团队空间创建、成员协同、实时编辑等功能。

**核心工作与技术贡献：**

1.  **高性能缓存优化：** **设计实现**基于 Redis + Caffeine 的多级缓存架构，**优化**主页热门图片查询接口，**显著提升性能**（接口响应时间**降低至原1/5**）。**引入**随机过期时间策略，**有效规避**缓存雪崩风险。
2.  **AI绘图服务集成：** **封装**阿里云百炼大模型，**提供**图片生成任务创建与查询API。**采用**异步任务轮询机制处理进度，**设计**统一错误码体系，**提升**服务稳定性与易用性。
3.  **实时消息与系统健壮性：** **基于** Disruptor 高性能无锁队列，**实现** WebSocket 消息的异步化处理，**大幅提升**系统吞吐量。**结合** `@PreDestroy` 注解**实现**服务优雅停机机制，**保障**用户编辑操作的完整性，**防止**数据丢失。
4.  **高并发空间管理：** **保障**线程安全与数据一致性，**设计实现**基于 `ConcurrentHashMap` 分段锁 + `TransactionTemplate` 编程式事务的空间创建服务，**严格限制**每个用户仅能创建一个私有空间。
5.  **以图搜图功能集成：** **实现**基于百度以图搜图 API 的图片检索功能。**运用** Jsoup + HttpClient **进行** API 调用，**采用**门面模式**封装**组合调用逻辑，**提供**简洁易用的图片列表接口供客户端调用。
6.  **高性能数据分片：** **应用** ShardingSphere **实现**团队空间图片数据的动态分表策略，**自定义**分表算法**提升**海量数据查询效率。**手动维护**可用分表节点，**确保**符合框架校验规则。
7.  **细粒度权限控制：** **基于** Sa-Token 权限框架，**采用**其 Kit 模式**设计实现**支持多账号体系的 RBAC 权限控制系统。**通过**统一拦截请求上下文参数**完成**权限校验逻辑，**运用**自定义注解合并**简化**鉴权注解使用，**轻松支持**方法级别的细粒度权限控制。
